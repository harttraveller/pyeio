    Overview

    Python Easy Input Output is a Python library meant to simplify file IO processes. The primary benefit is the ability to load and save different file formats with just the load and save methods.

    import json
with open("data.json", "r") as file:
    data = json.load(file)
file.close()

    from pyeio import easy
data = easy.load("data.json")
    Installation
    pip install pyeio
    Import
    from pyeio import easy

    Tutorial

    Save Files

    Input Dictionary
    data = {
    "apples": 3,
    "oranges": 4,
    "bananas": 5
}
    Python Code
    easy.save(data, "example.json")
    Output JSON File
    {
    "apples": 3,
    "oranges": 4,
    "bananas": 5
}

    Save Custom Format

    Input Dictionary
    data = {
    "apples": 3,
    "oranges": 4,
    "bananas": 5
}
    Python Code
    easy.save(data, "example.csv")
    Output CSV File
    apples, 3
oranges, 4
bananas, 5

    Load Files

    Input JSON File
    [1, 2, 3, 4, 5, 6]
    Python Code
    data = easy.load("example.json")
    Data & Type
    [1, 2, 3, 4, 5, 6] <class 'list'>

    Load Custom Types

    Additional Import
    from pyeio import kind
    Input JSON File
    [1, 2, 3, 4, 5, 6]
    Python Code
    data = easy.load("example.json", astype=kind.numpy.array)
    Data & Type
    [1 2 3 4 5 6] <class 'numpy.ndarray'>
